# fmcg-sales-analysis-powerbi
FMCG Sales Performance Analysis — Freshway International Trading, Qatar
A Power BI dashboard built to analyse salesperson and product-level sales performance for February and March, helping management monitor targets, track variance trends, and identify top and bottom performers.

Objectives

Compare actual sales vs target sales for each salesperson across February and March
Identify top and bottom performing salespersons through variance analysis
Analyse product-wise sales quantities and contributions per salesperson
Determine which salesperson sold the maximum quantity of each product

Performance Indicators (KPIs)
Total Sales · Total Target · Target Achievement % · Monthly Sales Variance · Sales Variance % · Month-over-Month Growth % · Average Sales by Salesman
Dashboard Features
Salesperson-level variance analysis · Product contribution breakdown · Actual vs target comparison · Month-over-month trend tracking · Interactive slicers for salesperson, product, and month
